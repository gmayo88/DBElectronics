To set up the database:
1) Import the file 'electronics_vendor.sql' into your database server.
2) Import the file 'useraccountprivileges.sql' to set up the user accounts. 
	All name and passwords are found in dblogin.txt
3) To set up the data, import the files found in the folder 'DB Data'
4) The web interface is designed to run on an AMP stack. 
	To set up the web interface, place the folders found in the Interfaces/Database Website folder
	into the docutment root folder of the running Apache server.
5) The call center interface is a Java files.
